# Sistema-Bibliotecario

Este es un proyecto de sistema bibliotecario desarrollado en C++. Está diseñado para ser utilizado por bibliotecarios para gestionar libros, usuarios y transacciones de préstamo y devolución.

## Estado Actual del Sistema

El sistema se encuentra en una etapa inicial de desarrollo. Actualmente cuenta con:

- Un menú principal que presenta las opciones disponibles.
- Clases declaradas para Usuario, Usuarios y Devolucion, pero sin implementación completa.
- Funcionalidades planificadas pero no implementadas en los casos del menú.

### Funcionalidades Planificadas

1. **Registrar Libros**: Permitir agregar nuevos libros al sistema.
2. **Mostrar Usuarios**: Listar los usuarios registrados.
3. **Devolver Libro**: Gestionar la devolución de libros prestados.
4. **Prestar Libro**: Gestionar el préstamo de libros a usuarios.
5. **Reportes y Estadísticas**: Generar informes sobre el estado de la biblioteca.
6. **Salir**: Opción para cerrar el sistema.
